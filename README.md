📖 About the Project

The Client Enquiry CRM is a modern Customer Relationship Management (CRM) platform developed for Glory Simon Interiors to digitize and streamline client interactions, enquiry management, quotation generation, communication, reporting, and project monitoring.

The system replaces manual workflows with an intelligent web application that enables administrators to efficiently manage customers, site visits, quotations, reports, and AI-powered design recommendations through a single centralized dashboard.
